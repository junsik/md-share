# Changelog

## 1.0.0 — 2026-07-16

- 본문과 메타데이터의 원자적 파일 저장 및 고아 파일 정리를 추가했다.
- `Idempotency-Key` 기반 재시도 안전 업로드와 안정적인 오류 code를 추가했다.
- 원본 `.md` 파일명, UTF-8, NUL 문자와 2 MiB 제한 검증을 추가했다.
- 웹 편집기에 `.md` 파일 선택과 drag-and-drop을 추가했다.
- 한 번만 반환되는 문서 관리 token으로 만료 변경과 삭제를 지원한다.
- 공개 메타데이터, 안전한 저장 상태 API와 OpenAPI 3.1 계약을 추가했다.
- 저장소, API와 입력 검증 자동화 테스트를 추가했다.
